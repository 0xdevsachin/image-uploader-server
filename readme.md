* NodeJS

# Development Setup

```shell
# Clone this repository
git clone https://github.com/0xdevsachin/image-uploader-server.git

# Install dependencies
$ npm install

# Run the app
$ npm start
```

# Available Routes

* Get  "/ " To check if Server is Up
* Post "/uploads" To upload Images
